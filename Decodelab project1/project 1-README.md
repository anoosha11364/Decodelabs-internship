## 📌 Project Overview

Project 1 is the **Interface Phase** of the DecodeLabs Full Stack Internship. The goal was to build a fully responsive frontend interface using pure HTML, CSS, and JavaScript — no frameworks, just fundamentals.

---

## 🎯 Goal

Create a responsive frontend interface for a simple web application that works seamlessly on any device.

---

## ✅ Key Requirements

- Built with HTML, CSS, and basic JavaScript only
- Responsive layout for mobile, tablet, and desktop
- Clean and user-friendly UI
- No frameworks used

---

## 🛠️ Key Skills Demonstrated

- Semantic HTML5 structure
- CSS Grid and Flexbox for responsive layouts
- Mobile-first design approach
- JavaScript DOM manipulation
- WCAG accessibility standards
- 2025 UI/UX aesthetics
